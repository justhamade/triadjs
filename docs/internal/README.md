# Internal

These files are for framework contributors, not users. They are not linked from the main documentation index and should not be treated as authoritative guides.

## Contents

- **`prompts/`** — The original design prompts that bootstrapped Triad. Each phase of the framework was designed by feeding one of these prompts to an AI coding assistant. They are preserved here so future contributors can see the exact brief each subsystem was built from, diff the spec against the implementation, and understand the reasoning behind design choices that might otherwise look arbitrary in the code.

If you're a user looking for documentation, start at [`../README.md`](../README.md).
